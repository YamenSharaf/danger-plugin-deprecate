> allows to warn or prevent merge on unwanted changes

## Usage

Install:

```sh
yarn add danger-plugin-deprecate --dev
```

At a glance:

```js
// dangerfile.js
import deprecate from 'danger-plugin-deprecate'

deprecate()
```
